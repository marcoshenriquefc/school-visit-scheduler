# Leads Export

## Endpoints
- `GET /admin/leads/export/csv`
- `GET /admin/leads/export/pdf`

## Permissions
- ADMIN
- MARKETING
- COMMERCIAL

## Filters
- formId
- unitId
- gradeId
- status
- rubeusStatus
- startDate
- endDate

## CSV columns
- Name
- Email
- Phone
- Unit
- Grade
- Date
- Time
- Status
- Rubeus Status
- Created At

PDF export includes system header, filters and a tabular listing.
